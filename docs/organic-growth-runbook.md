# Xarchive Organic Growth Runbook

Last updated: 2026-04-06

## North Star

- Primary metric: `Organic visitors`
- Comparison window: compare the most recent 14 full days against the previous 14 full days
- Target: `+50% organic visitors by 2026-05-06`

## Weekly Operating Checklist

### Week 1

- Record the 14-day baseline for:
  - organic visitors by landing page
  - bounce rate by landing page
  - top referrers
  - top landing routes
- Confirm that the following URLs are live and indexable:
  - `/`
  - `/archive-instagram`
  - `/archive-reddit`
  - `/guides/instagram-archives`
- Submit all newly published URLs to Bing Webmaster Tools first.
- Submit the same URLs to Google Search Console as a secondary step.

### Week 2

- Publish the Instagram guide cluster.
- Check Bing indexing status for each new Instagram URL after 7 to 10 days.
- Review whether `/archive-instagram` is sending traffic deeper into the new guide cluster.

### Week 3

- Publish the Reddit expansion pages.
- Check Bing indexing status for each new Reddit URL after 7 to 10 days.
- Compare Instagram vs Reddit cluster performance by:
  - organic landing visits
  - guide-page bounce rate
  - guide-to-tool clickthrough

### Week 4

- Refresh the top Twitter pages that already attract search demand.
- Reallocate the final content slots to the better-performing cluster, defaulting to Instagram unless Reddit clearly wins.
- Identify any new URL that still has no traction after 10 days and improve internal links plus the opening answer block before publishing more pages.

## Bing-First Publishing Workflow

1. Publish the page and confirm it appears in the sitemap.
2. Open Bing Webmaster Tools URL inspection and submit the page directly.
3. Request indexing for the same page in Google Search Console, but do not make Google the blocking dependency.
4. Add at least one new internal link from:
   - the relevant tool page
   - the relevant hub page
   - one sibling guide
5. Re-check indexing after 7 to 10 days.

## Page-Level Acceptance Checks

- The first screen answers the query directly in plain English.
- The page includes:
  - quick answer
  - step-by-step workflow
  - common failure cases
  - FAQ
  - tool CTA
  - hub CTA
  - sibling workflow links
- The page has structured data where supported by the current template.
- The page can be reached from at least two other indexable pages.

## Decision Rules

- If Instagram guides produce more organic landings than Reddit by 2026-04-20, continue allocating new-page capacity to Instagram.
- If a page is indexed but does not attract clicks, tighten the title, opening answer block, and internal links before publishing more variants.
- If a page is not indexed after 10 days, add stronger internal links and resubmit in Bing before expanding that cluster further.
